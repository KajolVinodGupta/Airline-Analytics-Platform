# ✈️ Airline Analytics Platform

This project combines real-time flight tracking, delay analytics,
ticket sales insights, and ML forecasting using Streamlit, SQL, and Power BI.

## Features (Planned)

1. Real-Time Flight Tracker
2. Delay Analyzer
3. Machine Learning Delay Prediction
4. Sales Insights
5. Revenue Forecasting
6. PowerBI Dashboard
